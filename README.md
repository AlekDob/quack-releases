# 🦆 Quack Releases

Official releases for **Quack** - Multi-Agentic Desktop Application

## 📦 Downloads

Download the latest version of Quack for your platform:

### macOS
- **Apple Silicon (M1/M2/M3)**: Download the `.dmg` file with `aarch64` in the name
- **Intel**: Download the `.dmg` file with `x64` in the name

### Installation Instructions

1. Download the appropriate `.dmg` file for your Mac
2. Open the downloaded file
3. Drag Quack to your Applications folder
4. Open Quack from Applications

**Note**: On first launch, macOS may show a security warning. Go to **System Settings > Privacy & Security** and click **"Open Anyway"**.
