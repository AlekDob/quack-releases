## What is Quack?

Quack is a **desktop application for developers** that brings Claude's agentic capabilities to your local machine. Think of it as a visual IDE for AI agents - each with their own personality, tools, and memory.

### Key Features

| Feature | Description |
|---------|-------------|
| **Multi-Agent System** | Create specialized AI agents with unique personalities and roles |
| **Integrated Terminals** | xterm.js terminals with AI assistance built-in |
| **Kanban Board** | Visual task management for AI workflows (Cmd+K) |
| **Second Brain** | File-based knowledge store in `~/.quack/brain/` |
| **MCP Integration** | Connect Gmail, GitHub, Twitter, and 50+ tools |
| **Background Tasks** | Non-blocking execution of long-running operations |

### Why Quack?

- **Privacy First**: Your code and conversations stay on your machine
- **Multi-Project**: Work on multiple codebases with dedicated agents
- **Extensible**: Add any MCP server for custom integrations
- **Native Performance**: Built with Tauri for a fast, lightweight experience

## Downloads

### macOS (Universal Binary)

Works on both **Apple Silicon** (M1/M2/M3/M4) and **Intel** Macs.

| Format | Link |
|--------|------|
| DMG Installer | [**Download .dmg**](https://github.com/AlekDob/quack-releases/releases/latest) |

### Installation

1. Download the `.dmg` file
2. Open and drag **Quack** to your Applications folder
3. Launch Quack from Applications
4. **First launch**: Go to System Settings → Privacy & Security → Click "Open Anyway"

> **Note**: Quack is signed and notarized by Apple. The security prompt only appears on first launch.

---

## Auto-Updates

Quack includes **automatic updates**. When a new version is available, you'll see a notification inside the app. Updates are downloaded in the background and applied on next launch.

## Tech Stack

Built with modern technologies for performance and reliability:

- **Tauri 2.x** - Rust-based framework for native apps
- **React 19** - UI framework
- **TypeScript** - Type-safe code
- **Claude Agent SDK** - AI capabilities
- **xterm.js** - Terminal emulation
